## 2. Phase 1：RAW 上傳管線 + AI 調色

### 2.1 目標

建立可靠的 RAW 大檔案上傳管線、AI 基本調光調色、48 小時交付 SLA。這是 WanderLens 服務承諾的技術基石。

### 2.2 開發範圍

| 模組 | 內容 |
|------|------|
| 上傳服務 | 大檔案分段上傳（Multipart Upload）、斷點續傳 |
| JPEG 快路徑 | 機身 JPEG 優先處理 → AI 調色 → 相簿上架 |
| RAW 儲存 | 物件儲存（S3/GCS）、生命週期管理 |
| 批次驗收 | 檔案數、容量、格式、可解碼性驗證 |
| AI 基本調色 | 自動調光調色、批次處理 |
| 預覽圖產生 | 自動產生預覽圖、相簿縮圖 |
| 48h SLA 監控 | 倒數計時、失敗重試、告警通知 |
| API 介面 | 上傳 token 驗證、驗收回報、AI 狀態回報 |
| 儲存週期 | RAW 交付後最少 30 天、精修延長 |

### 2.3 成功判準

| 指標 | 意義 |
|------|------|
| RAW 上傳成功率 | 大檔案可靠上傳 |
| 斷點續傳可用 | 網路中斷可恢復 |
| JPEG 快路徑有效 | 不被大檔卡住交付 |
| 48h SLA 達成率 | 核心服務承諾 |
| AI 調色品質 | 基本水準可接受 |
| 失敗重試可靠 | 管線容錯 |
