## 2. Phase 1：網站獲客 + 首次下單

### 2.1 目標

讓使用者不必下載 App 也能理解服務、搜尋攝影師、完成首次下單。拍攝後以網頁版相簿過渡，App 留存價值待階段二再放大。

### 2.2 開發範圍

| 模組 | 內容 | JS 參考 |
|------|------|---------|
| 首頁 | Banner、服務介紹、拍攝類型卡片、預約步驟、攝影師展示、評價、景點、IG | joyshot-app Home.vue ⭐⭐⭐ |
| 搜尋列 | 拍攝類型、日期、地點（Google Places）、時段、時數、人數 | joyshot-app Header.vue ⭐⭐⭐ |
| 搜尋結果頁 | 攝影師卡片列表（作品集縮圖、評分、價格） | joyshot-app Match.vue ⭐⭐⭐ |
| 攝影師詳情頁 | banner、avatar、簡介、服務類別、特色、評價、作品集、預約按鈕 | joyshot-app Photographer.vue ⭐⭐⭐ |
| 預約流程 | Step 1-6（類型→配置→時間地點→選人→確認→結帳） | joyshot-app Checkout.vue ⭐⭐ |
| 結帳 | 客戶資料、明細、折扣碼、綠界付款 | joyshot-app Checkout.vue ⭐⭐⭐ |
| 付款結果 | 付款成功/失敗頁 | joyshot-app Thankyou.vue ⭐⭐⭐ |
| 網頁版相簿 | 基本相簿瀏覽、下載、分享連結（過渡用） | 無（新增） |
| FAQ | 常見問題 | joyshot-app Faq.vue ⭐⭐ |
| 隱私政策 | 隱私條款 | joyshot-app PrivacyPolicy.vue ⭐ |
| 多語系 | zh/en/jp/ka 四語系 | joyshot-app i18n ⭐⭐⭐ |
| SEO 基礎 | meta、Open Graph、hreflang | joyshot-app index.html ⭐⭐ |
| 分析追蹤 | GA、Meta Pixel | joyshot-app ⭐⭐⭐ |

### 2.3 超越 JS 的部分

| 項目 | JS | WanderLens |
|------|-----|------------|
| 預約流程 | 搜尋→看詳情→預約 | Step 1-6 結構化流程（類型→配置→時間地點→選人→確認→結帳） |
| 配置選擇 | 無 | 拍攝配置（外拍/棚拍、單機/雙機、造型師） |
| 選到即確定 | 搜尋後預約 | 時段鎖定，選到即確定 |
| 相簿 | 無 | 網頁版相簿（過渡） |
| SEO 內容池 | 無 | 預留 SEO 頁架構（階段三啟動） |
| 站內溝通 | 無 | 訂單成立後可進入站內溝通室 |

### 2.4 成功判準

| 指標 | 意義 |
|------|------|
| 訪客→預約開始率 | 獲客頁是否有效 |
| 預約→付款完成率 | 流程摩擦 |
| 首次下單不需 App | 網站獨立完成轉換 |
| SEO 基礎可被索引 | 長期獲客基礎 |
