## 2. Phase 1：工單協作基本功能

### 2.1 開發範圍

| 模組 | 內容 |
|------|------|
| 登入 | 外包公司帳號登入 |
| 工單列表 | 待處理、進行中、已完成、退修 |
| 工單詳情 | 照片清單、修圖規範、客戶備註、交期 |
| RAW 下載 | 取得指定 RAW（不開放整份訂單） |
| 成品上傳 | 上傳精修成品、格式驗證 |
| 狀態回報 | 接受/進行中/完成/退修 |

### 2.2 成功判準

| 指標 | 意義 |
|------|------|
| 工單可流轉 | 接收→處理→交付完整 |
| RAW 安全下載 | 僅取得指定照片 |
| 成品可驗收 | 格式/張數/品質可控 |
