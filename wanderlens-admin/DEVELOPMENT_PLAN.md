## 2. Phase 1：營運後台基本功能

### 2.1 開發範圍

| 模組 | 內容 | JS 參考 |
|------|------|---------|
| 登入 | 管理員登入 | joyshot-admin Login.vue ⭐⭐⭐ |
| Dashboard | 統計圖表、使用紀錄、快捷選單 | joyshot-admin Dashboard.vue ⭐⭐⭐ |
| 使用者管理 | 帳號 CRUD、角色指派 | joyshot-admin users/maintain ⭐⭐⭐ |
| 角色管理 | RBAC、選單權限 | joyshot-admin roles ⭐⭐⭐ |
| 攝影師管理 | 列表、詳情 7 分頁、上架開關、時段主控台 | joyshot-admin photographers ⭐⭐⭐ |
| 客戶管理 | CRUD、匯入匯出 | joyshot-admin customers ⭐⭐⭐ |
| 訂單管理 | 5 狀態分頁、後台手動建立、訂單歷程 | joyshot-admin orders ⭐⭐⭐ |
| RAW 驗收 | 批次上傳狀態、檔案數/格式/容量驗收 | 無（新增） |
| AI 交付監控 | AI 狀態、48h SLA 倒數、失敗告警 | 無（新增） |
| 清算撥款 | 訂單金額、抽成、應付、撥款紀錄 | joyshot-admin（擴展）⭐⭐ |
| 客服與爭議 | 訂單事件、專案文件、溝通調閱、退款 | 無（新增） |
| 優惠券管理 | 折扣碼 CRUD、使用追蹤 | joyshot-admin coupons ⭐⭐⭐ |
| 推廣員管理 | 銀行資訊、統計報表 | joyshot-admin affiliate ⭐⭐⭐ |
| 行政區域 | 樹狀 CRUD、多語系、最低時數 | joyshot-admin areas ⭐⭐⭐ |
| 服務項目 | 拍攝類型 CRUD | joyshot-admin services ⭐⭐⭐ |
| 內容管理 | FAQ、IG、Banner、News、景點 | joyshot-admin ⭐⭐⭐ |
| LINE 群發 | 攝影師群發訊息 | joyshot-admin broadcast ⭐⭐⭐ |
| 站內溝通調閱 | 溝通室調閱、調閱日誌 | 無（新增） |

### 2.2 超越 JS 的部分

| 項目 | JS | WanderLens |
|------|-----|------------|
| RAW 驗收 | 無 | 批次驗收、檔案完整性 |
| AI 監控 | 無 | 48h SLA 倒數、失敗告警 |
| 溝通調閱 | 無 | 站內溝通完整記錄調閱 |
| 爭議處理 | 無 | 爭議流程、退款、裁決 |
| 清算帳本 | 轉帳紀錄 | 完整分項帳本 |

### 2.3 成功判準

| 指標 | 意義 |
|------|------|
| 訂單全生命週期可監控 | 營運可控 |
| RAW 驗收可視化 | 交付管線健康 |
| 48h SLA 可監控 | 服務承諾可控 |
| 溝通可調閱 | 爭議可處理 |
| 清算可對帳 | 金流可控 |
