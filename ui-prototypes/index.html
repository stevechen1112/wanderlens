如何使用
  • 點下方卡片進入各原型；頂部工具列可開啟「設計說明」、切換手機預覽
  • 請用 Chrome / Edge 直接開啟 HTML；若在 IDE 內嵌預覽出現 file:// frame 安全警告，可忽略或改用瀏覽器開啟
  • 紅框／紅標 = 現站 WanderLens 問題；綠框／綠標 = 建議方向
  • 所有資料為 demo（Lens 大師 UUID 等），不呼叫 production API
  • 確認方向後，再指定要落地至 wanderlens-web / wanderlens-provider 的頁面
